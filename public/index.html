Документация

/v1/

  • выводит приветствие пользователю

/v1/comments

  • GET:возвращает список комментариев в формате json.
  • POST: возвращает обновленный список комментариев в формате json.

/v1/stats

  • GET: возвращает небольшой фрагмент в виде html таблицы, где в одной колонке user-agent, а во второй количество запросов.
  • POST: выводит сообщение о невозможности поддержки метода.
